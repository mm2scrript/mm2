## 🛠️ Troubleshooting Common Issues

**❌ Script does not load after execution**
The Mm2 Script may fail to attach if your executor version is outdated or incompatible with the latest Roblox update. **Solution:** Update your executor to the newest release, restart Roblox, and re-inject the Mm2 Script loader from a clean session.

**❌ ESP labels appear in wrong positions**
Mm2 Script Esp sometimes renders offset labels when Roblox's camera FOV is set to non-default values. **Solution:** Reset your Roblox camera FOV to `70` in settings, then reload the Murder Mystery 2 Script Esp module from the GUI.

**❌ Auto Collect teleports to empty spots**
Mm2 Script Auto Collect Coin can target coin positions that have already been picked up by another player due to network latency. **Solution:** Lower the `collect_delay` value in the config to `0.1` so the script re-scans positions more frequently.

**❌ GUI does not appear after injection**
The Mm2 Script Gui may be hidden behind the Roblox chat window or inventory panel. **Solution:** Press your configured GUI keybind (default `RightShift`) twice to force a re-render, or close all Roblox UI panels first.

**❌ Auto Shoot fires at teammates**
Mm2 Script Auto Shoot Murd may misidentify roles during the first two seconds of a round before role data fully loads. **Solution:** Enable the `role_check_delay` option in the Mm2 Script Admin Panel to add a brief verification pause before firing.

**❌ Freeze Trade does not lock the window**
Murder Mystery 2 Script Freeze Trade requires the trade window to be fully open before activation. **Solution:** Wait until both players have the trade GUI visible, then toggle the freeze option inside the Mm2 Script Gui.
